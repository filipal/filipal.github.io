In this project, I've explored two unique game concepts: "Spellbound Chronicles" and "String". "Spellbound Chronicles" is a vertical slice of a 2D side-scrolling platformer inspired by Croatian and Balkan mythology. Created using Unity, I was responsible for all aspects except the audio. The game features intricate sprites and concept art, reflecting the rich cultural heritage of the region. "String", on the other hand, is an original game concept presented as a 2D vertical slice. I created all assets and animations, with the code provided by my lecturer. This game emphasizes the creative potential of simple 2D design, showcasing various sprites and assets. Both projects highlight my ability to blend cultural inspiration with technical skill, creating immersive and visually engaging game experiences.
